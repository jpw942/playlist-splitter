# Day 3 Notes — Embeddings

## What is an embedding? Why is the distance between two embeddings meaningful?

An embedding is essentially assigning a vector containing numerical values to a piece of data (e.g. words, text, sound, etc). This is useful for machines because it allows them to be able to classify and recognize the data that is presented to it. Each feature in the vector allows the machine to graph the data which can help it observe it and the other points around it. The distance between two embeddings is meaningful because it shows how closely two pieces of data are related. For example, the distance between two words in a given text 'baseball' and 'football' may be small, which shows that these two words are more closely related.
